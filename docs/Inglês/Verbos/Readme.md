## `To` + Verbo (Infinitivo)

\* Quando o verbo começar com `to` quer dizer que o verbo está no infinitivo, ou seja o verbo sem conjugação.

+ He doesn’t want **to travel** with them.
  `Ele não quer viajar com eles/elas.`
