# Operadores

### Operadores de Atribuição
```csharp
int i = 1;

i++; // incremento de 1
i--; // decremento de 1

i += 2; // incremento de 2;

i = 1;

int x = i++; // x = 1 | i = 2 | seta de i em x e depois incrementa i
int x = ++i; // x = 2 | i = 2 | incrementa i e depois seta novo valor em x

i = 1;
int y = 1; 
y += i; // y = y + 1

int z = 2;
z *= i; // z = z * i = 2

```
